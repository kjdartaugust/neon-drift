# NEON DRIFT

A neon arcade space shooter that runs in any browser — desktop or phone. No build step, no dependencies: one self-contained `index.html`.

## Play

- **← → / A D** — move
- **Space** — shoot
- **P** — pause
- **Mobile** — drag to move, hold to fire

Grab the **⚡** power-up for 6 seconds of rapid fire. Difficulty ramps every wave, and your best score is saved locally.

## Features

- Canvas rendering with particles, screen shake, and parallax starfield
- WebAudio sound effects (generated at runtime, no audio files)
- Two enemy types, wave-based difficulty scaling
- Keyboard + mouse + touch controls
- Local best-score persistence

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```
